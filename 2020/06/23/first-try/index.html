mail team python base image Project

该项目关联nobody公司domeos并用来构建邮件组python基础镜像

说明

  • 安装rsyslog兼容以前日志收集
  • 安装haproxy
  • python2版本2.7.5
  • python3版本3.6.7
  • supervisor安装在python2版本之下

注意

  • python2版本命令为python 和 pip
  • python3版本命令为python3 和 pip3
  • pypi源使用公司大数据中心维护私有源，地址为 http://pypi.org/pypi/web/simple; 如果修改系统pypi源配置，见项目根目录下pip.conf，可放置于镜像的~/.pip/pip.conf目录下

增加logstash基础镜像

由于大数据中提供的ELK资源暂时没有提供logstash服务器，我们暂时把logstash服务放到我们这里来。

说明

  • logstash在项目里启动时不用指定配置文件，使用默认的pipelines.yml配置文件
  • 要新增项目的文件以logstash_***.conf命名,***为项目名,然后将该文件以新pipeline的方式include到piplelines.yml里
